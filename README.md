📊 HR Analytics Dashboard (Power BI)
🚀 Project Overview
This project presents an interactive HR Analytics Dashboard built using Microsoft Power BI to analyze employee data and identify key HR insights such as attrition, performance, and workforce trends.
The dashboard helps HR teams make data-driven decisions to improve employee retention and organizational efficiency.

🎯 Objectives
* Analyze employee attrition trends
* Identify high-risk employees
* Track workforce demographics
* Monitor performance and satisfaction metrics
* Support strategic HR decision-making

📂 Dashboard Pages
* Home Page
Overall employee summary
Key KPIs (Total Employees, Attrition Rate, Avg Salary)
* Action Page
High-risk employees identification
Department-wise attrition analysis
Filters for dynamic insights
* Details Page
Employee-level detailed data
Drill-through analysis
